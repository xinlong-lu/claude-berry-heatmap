# 🫐 Wild Berry Map USA

An interactive web map showing wild berry sightings across the United States, powered by iNaturalist data.

## Project Structure

```
berry-map/
├── index.html        ← The map web app (open this in a browser)
├── berries.json      ← Berry sighting data (generated by the Python script)
├── fetch_berries.py  ← Python script to fetch fresh data from iNaturalist
└── README.md
```

## How to Use

### 1. View the map locally
Just open `index.html` in your browser. It reads from `berries.json` automatically.

### 2. Refresh the data (optional)
Run the Python script to fetch the latest sightings from iNaturalist:

```bash
pip install requests
python fetch_berries.py
```

This overwrites `berries.json` with fresh data. Then refresh your browser.

### 3. Deploy to GitHub Pages (free live website)
1. Create a new GitHub repository
2. Upload all files in this folder
3. Go to Settings → Pages → Source → Deploy from branch → main
4. Your site will be live at `https://yourusername.github.io/berry-map`

## Features
- 📍 **Marker view** — individual sighting pins with popups
- 🌡 **Heatmap view** — density visualization
- 🫐 **Berry filter** — toggle species on/off
- 🌸 **Season filter** — filter by spring/summer/fall
- 📖 **Berry guide** — identification info and peak seasons

## Adding More Berries
Open `fetch_berries.py` and add entries to the `BERRY_SPECIES` dictionary.
Find taxon IDs by searching https://www.inaturalist.org/taxa

## Tech Stack
- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Maps**: Leaflet.js + Leaflet.heat plugin
- **Tiles**: CartoDB Dark Matter (free)
- **Data**: iNaturalist API (free, no key required)
- **Backend/Data**: Python + requests
- **Hosting**: GitHub Pages (free)
